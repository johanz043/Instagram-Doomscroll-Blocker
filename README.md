# Instagram-Doomscroll-Blocker
Blocks posts from your main feed, explore page and reels page. Only allows you to access messages and stories. Works on Google Chrome.

## Download Instruction
**Step 1** - Click code then download the zip file.

**Step 2** - Extract the zip file into a path of your choosing

**Step 3** - Go to Google Chrome

**Step 4** - Go to extensions
```bash
chrome://extensions/
```

**Step 5** - Turn on Developer Mode, it should be on the top right

**Step 6** - Click Load Unpacked and select the Doomscroll Blocker file
